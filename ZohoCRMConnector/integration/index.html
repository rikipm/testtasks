Перед использованием приложения необходимо произвести интеграцию.
Сначала зарегистрируйте приложение здесь
В строке "Имя клиента" введите любое имя. В логах им будут подписаны все действия этого прииложения
В строке "Домен клиента" введите домен по которому будет распологаться сервис (Например https://example.com)
В строке "Авторизованные URL-адреса для перенаправления" укажите ваш домен + integration/handler.php (Например https//example.com/integration/handler.php) Выберите тип WEB-BASED
Вы получите ID и секрет клиента. Введите их ниже, а так-же введите URL адрес для перенаправления
